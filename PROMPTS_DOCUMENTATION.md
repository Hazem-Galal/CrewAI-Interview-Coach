## üéØ Prompt Optimization Best Practices

### 1. Dynamic Variable Injection
All prompts incorporate:
- `{company}` - Target company name
- `{job_position}` - Specific role title
- `{job_description}` - Full JD for context
- `{interviewer}` - Interviewer name for personalization
- `{candidate_resume}` - Background for customization

### 2. Personality Consistency
Each agent maintains:
- Consistent expertise level (years of experience)
- Unique communication style
- Specific methodologies and frameworks
- Authentic professional background

### 3. Anti-Repetition Mechanisms
Implemented through:
- Explicit "DO NOT repeat questions" instructions
- Memory integration across tasks
- Context awareness of previous interactions
- Progressive depth guidance

### 4. Output Format Standardization
Each task specifies:
- Expected word count or structure
- Required components (bullet points, scores, etc.)
- Tone and style guidelines
- Specific examples to include
